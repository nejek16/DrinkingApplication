## Values & formulas
### Default loss of alcohol
* Per hour = bca-(bca * 0.015)
* Per minute = bca-(bca * 0.00025)
* Per minute = bca-(bca * 0.00000416666)
### Food
bca= alcohol level
* Full meal -> bca * 0.75 (effect lasts 3h)
* Medium meal -> bca * 0.85 (effect lasts 3h)
* Snack -> bca * 0.99 (multiple snacks avalible max. 6)
* Water -> bca * 0.998 (0.5 liter)
